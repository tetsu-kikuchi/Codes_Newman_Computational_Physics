# Computational Physics with Python (based on a textbook by Mark Newman)

## Overview

This repository is a collection of jupyter notebooks in which I explored topics and problems on computational physics.  Many of the notebooks are motivated by problems and exercises in the textbook "Computational Physics" by Mark Newman.  Moreover, I put together some notebooks digging deeper into topics that caught my interest.  The content is at an introductory level.  

These notebooks were originally written for my own study, but I am sharing them in the hope that they may help others who are just starting out in numerical computation or computational physics, as I once did. I have written them with an effort to make them as clear and accessible as possible for readers.  

I did not only write down codes to solve problems, but also provided explanation of motivations, what to learn, and interpretation of the results in each of the notebooks.  


## Repository Structure & File Naming  

The directory structure follows the chapter organization of Newman's textbook. Within each chapter directory, there are two categories of notebooks:

- Files starting with "example_" or "exercise_": solutions to the corresponding examples or exercises in Newman's textbook, often along with deeper explorations motivated by them.  

- Files starting with "note_": notes summarizing topics I investigated to gain a deeper understanding the topic of each chapter.
